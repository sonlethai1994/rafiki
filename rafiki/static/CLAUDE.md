# rafiki/static/

Frontend source assets. Compiled by webpack into `rafiki/static/webpack_bundles/`.

## Structure

```
static/
  css/           # Compiled CSS (do not edit manually)
  fonts/         # Web fonts
  images/        # Image assets
  js/
    project.js   # Main app JavaScript entry point
    vendors.js   # Third-party library imports (Bootstrap, Popper, etc.)
  sass/
    project.scss              # Main SCSS entry point
    custom_bootstrap_vars     # Bootstrap 5 variable overrides
  webpack_bundles/            # Compiled output (gitignored, generated by webpack)
```

## Adding styles

Edit `sass/project.scss` or create a new partial and import it:

```scss
// project.scss
@import "custom_bootstrap_vars";
@import "bootstrap/scss/bootstrap";
@import "partials/my-component";  // new partial
```

## Adding JavaScript

Import in `js/project.js`:

```js
import "./my-module";
```

For third-party libraries, add to `js/vendors.js`:

```js
import "my-library";
```

Install via npm:

```bash
npm install my-library
```

## Bootstrap customization

Override Bootstrap 5 variables in `sass/custom_bootstrap_vars` **before** importing Bootstrap. Full list of variables: https://github.com/twbs/bootstrap/blob/v5.1.3/scss/_variables.scss

## Static files in production

In production, `collectstatic` gathers all static files into `staticfiles/`. WhiteNoise serves them with compression and long-term caching headers. Webpack bundles are built during the Docker image build (multi-stage).

Do not commit `webpack_bundles/` or `staticfiles/` — both are generated.
